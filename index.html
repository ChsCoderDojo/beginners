Welcome to CoderDojo SF!

If you're checking this page out, there's a good chance that you've never programmed before. Or maybe you have and you're not sure if you have enough experinece to work on the lessons we create. Don't worry, we're here to help.

While CoderDojo SF aims to create lessons that are accessible to programmers of all skill level, it does help to have a little experience with basic programming syntax.
